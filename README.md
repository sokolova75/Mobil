# Название проекта
Соколова МПБ-301
Приложение-счётчик шагов с сохранением истории. Позволяет пользователю вручную добавлять шаги, просматривать статистику за предыдущие дни и сохранять данные локально на устройстве.

## Стек
Язык: Dart
Фреймворк: Flutter
Библиотеки:
shared_preferences (^2.2.2) – для локального хранения данных
intl – для форматирования дат
Архитектура: Стандартный Flutter (StatefulWidget + StatelessWidget)

## Быстрый старт
Установленный Flutter SDK (версия совместимая с sdk: ">=2.18.0 <3.0.0")
Android Studio / VS Code с Flutter-расширением
Эмулятор Android/iOS или физическое устройство
1.Клонирование и установка
# Клонируем репозиториц
git clone https://github.com/sokolova75/Mobil.git
cd Mobil
# Установка зависимостей
flutter pub get
2. Запуск приложения
flutter run
3. Сборка релизной версии
flutter build apk --release
